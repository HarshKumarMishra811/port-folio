# 💼 Personal Portfolio Website

A modern, responsive, and interactive portfolio website built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. This portfolio showcases my skills, projects, education, achievements, and contact information with smooth animations and a clean user interface.
